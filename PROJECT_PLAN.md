# Living History Web – Plan base

## Objetivo
Convertir la guía práctica en castellano en una web interactiva estática con estética de presentación editorial.

## Estructura
- 1 portada sin número
- 10 diapositivas numeradas (1–10)
- menú superior con acceso a cada diapositiva
- navegación anterior / siguiente
- diseño responsive
- preparada para futuro selector de idioma

## Estilo
- inspiración en presentación tipo pitch deck/editorial
- titulares grandes
- jerarquía tipográfica clara
- bloques/caixas destacadas
- composición 16:9 inspirada en diapositiva clásica, sin tamaño fijo rígido

## Tecnología
- HTML, CSS y JavaScript simples
- sin frameworks
- apta para funcionar sola o dentro de un iframe

## Fase actual
- trabajar solo en castellano
- mantener interacciones existentes
- reorganizar y maquetar estructura global
- no cerrar todavía imágenes definitivas ni traducciones
